Midnighttigger's Default Connected Textures

    The Goal of this Resourcepack is to enable Default Connected Textures inspired by the default textures optifine has onto Fusion.
    With this & Fusion you can run default connected textures without continuity or optifine

Features?

  • Red/Normal Sandstone CT
  • Bookshelf CT
  • Glass (including tinted & stained) CT
  • Glass Panes (including tinted & stained) CT
  • "CT" Stands for Connected Textures
  • Addons such as Programmer Art and Clear Glass can be found on MT-CT-D-A

FAQ

  • No connected textures? -> look at the top of the modpage!
  • "made for an older version of Minecraft" -> just load the pack as normal and it should run fine unless mojang has made significant changes to their resourcepack format
  • Make sure you have the correct version for your version and correct version of Fusion
  • Addons? -> Right here: MT-CT-D-A
  • Breaks with sodium without indium, this is mostly because sodium (not embeddium) changes the fabric rendering system so indium fixes that
  • Use in Modpacks is specified in the license (dont worry the license should be easy to read)

Why was CTM & Athena Compat Depricated?

  • Athena does not support some features required to make this
  • CTM mod has not updated in a while and is limited up to 1.19.2 and lower
  • This Pack has became Fusion only because 1.Can make fusion install automatically when you install this pack, 2.Ease of only having to maintain a resourcepack in 1 format and 3.Most people using this pack use fusion with it
